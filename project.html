Food order website
This Food Order Website is a static web page that displays a list of food items and allows users
to place orders by clicking the "Order Now" button next to each item. The menu items are generated
dynamically using JavaScript, and when a user clicks the button, an alert message informs them of their order. The HTML file provides the basic structure of the website. It includes a title, references to CSS and JavaScript files, and a element with the id menu where the food items are dynamically added. Bootstrap CSS framework is included from a CDN to facilitate responsive design. The CSS file contains styles for the website elements. It styles the body, sets the font-family, and defines the appearance of menu items and the order button.
